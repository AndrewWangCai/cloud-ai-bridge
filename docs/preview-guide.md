# 网页预览指南

在沙箱里跑起 dev server 后，可以用临时链接在手机/电脑上预览你的网页。

## 步骤
1. 在终端起一个服务，比如：
   ```bash
   python3 -m http.server 3000
   # 或
   npm run dev        # Vite/Next 等
   ```
2. 切到工具栏【预览】。平台会自动检测容器内**正在监听的端口**并列出。
3. 选中端口即可在页内预览。

## 分享给别人看
- 点【🔗 分享】生成一个带 token 的临时链接：`/preview/<项目>/<端口>/?pt=<token>`。
- 链接**会过期**（默认 30 分钟），过期需重新生成。
- 每个项目最多同时有限个有效分享链接（超出会顶掉最旧的）。
- 拿到链接的人凭 token 访问，**不需要**你的账号；你自己凭登录会话访问。

## 为什么预览会打不开 / 过期
- **dev server 没起**：端口没人监听，预览自然 404。先确认服务在跑。
- **端口没被检测到**：平台只暴露容器内监听的端口；换个常见端口（如 3000/5173/8000）再试。
- **链接过期**：重新点【分享】。
- **容器离线**：先【激活并连接沙箱】。

## 安全说明
- 预览是受控反向代理：只代理**你这个项目容器**允许的端口，不能代理任意 URL，也访问不到宿主机或内网。
- 管理员可一键撤销所有预览链接。

## 限制
- Linux 宿主下预览直连容器 IP，正常工作；Windows/Mac 的 Docker Desktop 宿主无法直接路由容器 IP，本原型预览功能在那种环境下不保证可用。
