# 常见问题 FAQ

## 账号 / 注册
**Q: 怎么注册？还要邀请码吗？**
A: 不用邀请码了。填用户名+密码+**邮箱**注册，邮箱会收到 6 位激活码，输入激活后即可登录。同一网络每天注册次数有限。

**Q: 没收到激活邮件？**
A: 先看垃圾箱；仍没有就点【重发激活码】（有频率限制）。激活码 15 分钟内有效。登录时若提示「邮箱未激活」，也会让你补激活。

**Q: 忘记密码了怎么办？**
A: 登录页点【忘记密码？】→ 填注册邮箱 → 收 6 位验证码 → 输验证码 + 新密码即可重置。重置后旧的登录会话会全部失效，用新密码重新登录。（验证码 15 分钟有效；为防枚举，无论邮箱是否存在都提示“已发送”。）

**Q: 重启服务后要重新登录吗？**
A: 不需要。会话已持久化，conduit 重启不掉登录（除非你清了 cookie 或会话过期）。

**Q: 提示「该网络已被封禁」？**
A: 你的 IP 被风控封禁，联系管理员。

## 沙箱 / 激活
**Q: 一直「离线」/激活失败？**
A: 看页面给出的**具体失败原因**（auth/quota/workspace/docker/agent/网络 哪一步）。常见是服务器重启后容器未自启——重新点【激活并连接】。

**Q: 服务器重启后容器为什么停了？**
A: 容器用 `restart=no`（交给生命周期管理），重启后点一次【激活并连接】即可拉起。

**Q: 能同时开几个项目？**
A: 免费档同时 1 个运行中、每天创建 1 个。

## AI 工具
**Q: 我用的是 DeepSeek，它却说自己是 Claude？**
A: 那是模型在模仿所运行 CLI 的系统提示，不代表你在用 Claude。你用的就是你绑定的模型。详见 [ai-cli-guide.md](ai-cli-guide.md)。

**Q: 对话模式能像 Claude/Codex 一样弹 yes/no 吗？**
A: 终端模式保留交互式 yes/no；对话模式是 headless，自动执行（沙箱隔离下相对安全）。

**Q: 1C1G 为什么会卡 / OOM？**
A: 1 核 1G 适合小型 React/Vite/Express/FastAPI、静态站。Next.js build、Java/Rust 大项目、Playwright、多个 AI 同时跑容易爆内存。一次跑一个，必要时等付费扩容。

## 代码 / 数据
**Q: 我的项目/网页「没了」？**
A: 可能到期被销毁，或 AI 只是没有上次记忆（文件还在，`ls` 看看）。一定要[及时 git push](git-guide.md)。

**Q: 项目到期了还能找回吗？**
A: 备份保留期内可【恢复】或【下载】；超过保留期就彻底销毁了。见 [expiration-backup.md](expiration-backup.md)。

**Q: 怎么让 AI 知道我在做什么项目？**
A: 【配置】→「项目说明」写一句，会写入 `CLAUDE.md`。

## 预览
**Q: 预览打不开？**
A: 确认 dev server 在跑、端口在监听、容器在线、分享链接没过期。见 [preview-guide.md](preview-guide.md)。

## 限制 / 风控
**Q: 项目突然被停，原因写着「出站流量超限/超配额」？**
A: 触发了防滥用限制（爬虫/大下载/大文件）。这是保护服务器，避免账单失控。

**Q: 终端黑屏？**
A: 刷新页面，多为网络抖动。
