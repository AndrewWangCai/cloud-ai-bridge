# 社区：把你的项目公开给大家观赏

你可以自愿把某个项目发布到社区，让其他登录用户观赏；也可以选择是否允许别人「获取」（下载/拉取）。

## 怎么发布
1. 【配置】抽屉 →「发布到社区」。
2. 填**标题**、**一句话介绍**，选公开级别：
   - **仅观赏**：别人能看，但不能下载/拉取你的代码。
   - **可获取**：别人可以下载快照拿到你的项目。
3. 点【🌃 发布】。发布时系统会对你当前工作区做一份**只读快照**——所以**即使沙箱到期/停止，作品依然能被观赏**。
4. 想更新展示内容：改完再点一次【发布】即可覆盖快照。想撤回点【撤下】。

> 只有**项目所有者本人**（已登录 + 邮箱已激活）能发布自己的项目，别人无法替你公开。

## 别人看到什么
- 在 [社区页](/community.html)（或【逛社区】按钮）浏览所有公开作品。
- 「观赏」在一个**沙箱化 iframe** 里打开你的快照（带 `CSP: sandbox`，即使作品里有脚本也跑在隔离环境，碰不到别人的账号）。
- 「可获取」的作品多一个【下载】按钮，拿到 `.tar.gz` 快照。

## 隐私与安全须知
- 发布 = 把你工作区当时的文件做成公开快照。**发布前请确认目录里没有密钥、`.env`、隐私数据**——别把不该公开的东西一起发出去。
- 浏览社区**需要登录**（不是任何人都能看）。
- 管理员可对违规作品**一键下架**。
- 「仅观赏」不提供下载/克隆入口；但任何能在浏览器里渲染的内容，技术上都无法 100% 防止别人截图/抄写——它防的是「便捷获取」，不是「绝对不可见」。

## 给管理员
- 后台 `/admin.html` →「社区作品」可查看 / 打开 / **下架**任意作品（删除快照 + 取消公开）。
- 快照存于 `AI_SANDBOX_COMMUNITY_DIR`（默认 `conduit/community`，部署建议指到 `/opt/ai-sandbox/community`）；该目录已 gitignore，不入库。
