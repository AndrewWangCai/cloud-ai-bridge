# 新手上手：5 分钟跑通第一个项目

欢迎使用 **Cloud AI Bridge** —— 手机当遥控器，在云端隔离沙箱里跑 Claude Code / Codex / DeepSeek，写代码、测网页、推 Git。

## 1. 注册登录（邮箱激活）
1. 打开平台网址（管理员会给你）。
2. 点【注册一个】，填用户名、密码、**邮箱**。
3. 系统会把一封含 **6 位激活码**的邮件发到你邮箱（留意垃圾箱）。
4. 在激活框输入激活码 →【激活并进入】。下次直接用用户名 + 密码登录。
5. 没收到？点【重发激活码】（有频率限制）。

> 风控：同一网络每天最多注册有限个账号；激活码 15 分钟内有效。

## 2. 新建项目
1. 右上角点【配置】打开抽屉。
2. 在「项目」区输入一个名字（字母/数字/下划线/连字符），点新建。
3. **每个项目 = 一个独立的 Docker 沙箱**，1 核 1G，互不影响。免费档每人同时 1 个运行中项目、每天创建 1 个。

## 3. 绑定模型账号（二选一）
- **API Key**：在「模型 / 账号」填 Anthropic / OpenAI / DeepSeek 的 Key，点【保存并注入】。用量走你自己的账号。
- **会员号**：先【激活并连接沙箱】，在「模型 / 账号」点击 Claude/Codex 登录，到终端完成设备码登录；完成后回到设置里点“已完成”（凭证持久化，跨重启保留）。

详见 [ai-cli-guide.md](ai-cli-guide.md)。

## 4. 激活并连接
点【激活并连接沙箱】。系统会为你创建容器并连上：
- 状态变「在线」即可用；
- 若失败会显示**具体原因**（不是笼统的 500），按提示重试。

## 5. 开始干活
- **对话模式**：直接在聊天框发指令，AI 像 Claude/Codex 一样回你。
- **终端模式**：跑 `npm install`、`npm run dev`、`python3 -m http.server` 等。
- **预览网页**：起好 dev server 后切【预览】，详见 [preview-guide.md](preview-guide.md)。
- **保存代码**：用 [Git 助手](git-guide.md) 一键复制命令推到你自己的仓库。

## 6. 到期与备份
免费项目默认 **48 小时**后自动停止并备份，备份再保留一段时间（其间可下载 / 恢复 / git push）。详见 [expiration-backup.md](expiration-backup.md)。
