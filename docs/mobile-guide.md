# 手机使用指南

本平台是**手机优先**设计的。用手机浏览器即可远程控制云端 AI 编程环境。

## 打开方式
- 用手机浏览器访问平台网址（建议 Chrome / Safari）。
- 登录后界面与 Claude App 风格一致：上方工具栏切换「对话 / 终端 / 预览」，右上角【设置】是抽屉。

## 对话模式（推荐手机用）
- 像微信聊天一样发指令，AI 回成气泡。
- 适合「帮我写个登录页」「修一下这个报错」这类自然语言任务。
- 危险/敏感操作仍会让你确认。

## 终端模式
- 内置网页终端（xterm），可直接敲命令。
- 手机敲命令不便时，多用对话模式让 AI 代劳。
- 终端支持逐字回显与窗口自适应。

## 断线与重连
手机切后台、锁屏、切网络都可能让连接中断：
- 回到页面后，若显示「离线」，点【激活并连接沙箱】重新连上即可。
- **代码不会丢**：文件存在云端 workspace，断的只是连接，不是数据。
- 服务器重启后容器不会自启（设计如此），重新点一次【激活并连接】即可。

> 提示：无痕/隐私模式下 cookie 可能被清，会更容易要求重新登录。

## 省流量小技巧
- 预览大型页面会走流量；看完及时关。
- 容器出站流量有上限，跑爬虫/大量下载可能被自动停（防滥用）。

## 常见手机问题
- **终端黑屏**：刷新页面；多为网络抖动导致资源没加载完。
- **输入法挡住终端**：切到对话模式，或横屏。
