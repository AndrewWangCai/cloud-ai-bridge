安全与隐私
隔离优先，凭证私有
  • 每项目独立容器，cap-drop / no-new-priv / 限额 / 专网
  • 禁 privileged / host 网络 / docker.sock
  • API Key 仅注入你的容器，浏览器拿不到
  • 跨用户越权 / 路径穿越防护，日志脱敏
  • 出网加固：禁云元数据 / 内网 / SMTP 滥用
  • 到期自动备份、超期销毁，数据可控
