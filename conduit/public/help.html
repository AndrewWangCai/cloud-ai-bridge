使用帮助

① 注册 / 登录

  1. 点【注册一个】，填用户名、密码、邮箱。
  2. 邮箱会收到 6 位激活码（留意垃圾箱），输入激活即可登录。
  3. 忘记密码：登录页点【忘记密码？】→ 填邮箱收验证码 → 设新密码。

② 建项目 · 连接

右上角【设置 → 项目】新建一个项目（= 一个独立的云端沙箱），点【激活并连接沙箱】。每个免费项目 1 核 1G。

只要你常来用，项目到期时间会自动顺延，不会把在用的数据删掉。

③ 绑定模型 Key（最简单）

【设置 → 模型/账号】填任一家的 API Key，保存即用，用量走你自己的账号：

  • Claude（Anthropic）、Codex（OpenAI）
  • DeepSeek / GLM 智谱 / Kimi Moonshot / 通义 Qwen：都是填 Key 即用，对话经内置路由调用（OpenAI 兼容）。

在对话区上方下拉选平台 + 选模型即可切换。没绑 Key 时发消息会先提示，不会报底层错。

🌏 国内/香港部署推荐用国产模型（DeepSeek/GLM/Kimi/通义）——稳定无地区限制；Claude/Codex 可能因 Anthropic·OpenAI 的地区限制连不通。

④ 对话模式 vs 终端模式

对话模式（推荐手机用）：像聊天一样发指令，AI 在云端写代码、回成气泡。点【🧹 新对话】可清空并开启新一轮（不再延续上文）。

终端模式：内置网页终端，直接敲命令（npm/python 等）。点一下黑色区域弹出键盘。

⑤ 会员号登录（订阅用户，进阶）

有 Claude/Codex 订阅、不想用 API Key 的话可以走会员号登录：到终端运行登录命令、按提示在浏览器授权。

⚠️ 远程沙箱里跑 OAuth 登录较繁琐、偶有不稳。最省心还是用 API Key（③）。

⑥ 保存到你自己的 Git

沙箱是临时的，记得把代码推到自己的仓库：

  • Git 助手（设置 → Git）：填仓库地址 → 一键复制 git 命令到终端执行。
  • GitHub 一键（若管理员已配 OAuth）：连接 GitHub → 一键建仓 / 提交并开 PR。

⑦ 网页实时预览

在终端起服务（如 python3 -m http.server 3000 或 npm run dev）→ 切【预览】。可切换端口、输路径跳转、刷新、新窗口打开、生成可分享的临时链接（会过期）。

⑧ 多人协作

房主在【设置 → 协作/社区】邀请协作者；对方在项目下拉看到「👥协作」并加入，共享同一沙箱/终端/预览。

文件锁：在协作条填文件名【🔒锁】，别人就锁不了同一文件，各自的 AI 也会被告知不要改别人锁的文件，避免互相覆盖。

⑨ 社区

愿意的话把项目【🌃发布】到社区供大家观赏，可选「仅观赏」或「可获取（允许下载）」。发布前确认目录里没有密钥/隐私文件。仅本人可发布自己的项目。

⑩ 到期 / 备份 / 销毁

免费项目默认 48 小时到期（活跃会自动续期）→ 自动停止并备份 → 保留期内可下载 / 恢复 / 推到 Git → 超期彻底销毁。重要代码请及时 git push。

常见问题

用 DeepSeek 它却说自己是 Claude？

那是模型在模仿所运行 CLI 的提示词，不代表你在用 Claude。你用的就是你选的模型。

1 核 1G 会不会卡？

适合小型 React/Vite/Express/FastAPI、静态站。Next.js build、大型 Java/Rust、同时跑多个 AI 容易爆内存，建议一次跑一个。

终端打不了字？

用手指点一下黑色终端区域，软键盘会弹出来。

我的项目怎么"没了"？

可能到期被清理（活跃会续期），或只是 AI 没有上次记忆（文件还在，ls 可见）。务必及时 git push。

没收到激活/验证码邮件？

先看垃圾箱；仍没有就点重发（有频率限制），15 分钟内有效。
